## [1.0.5] - 2026-01-22

### Added
- `cs rename <old-name> <new-name>` command to rename cheatsheets
- `-f/--force` option for rename to overwrite existing sheets
- Sample cheatsheets in `sample/en/` and `sample/ja/` directories
  - vim.md - Vim commands
  - tmux.md - tmux commands
  - cc-skills.md - Claude Code skills
  - cs-help.md - This CLI help
